# VYOM-AI 🛡️
## AI-Powered Product Safety Intelligence Platform

A production-ready platform to ensure safety in Food, Skincare, Supplements, and Produce.

### 🚀 Quick Start
1. Clone the repo
2. Run `pnpm install`
3. Configure `.env` (see `.env.example`)
4. Run `pnpm dev`

### 🏗️ Architecture
- **Frontend**: Next.js 15, Tailwind CSS, ShadCN UI
- **Backend**: NestJS, Prisma, PostgreSQL
- **AI Service**: FastAPI, Groq LLM (RAG-first)

### 🔑 API Keys Needed
- Groq API (Llama 3)
- Mapbox GL
- Google Custom Search (optional)
- OpenFoodFacts (Public)
